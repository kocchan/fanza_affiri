# 大喜利回答ジェネレーター（坊主リプ集客）

坊主のような大喜利募集アカウントに返す**面白い大喜利リプ**を、ユーザーが入れた**お題**からその場で作る。
狙いは「リプが伸びる→プロフがクリックされる→美女系メインへ流入」という**集客ファネルの入口**。
このスキルは**回答文を作るだけ**。リプ・投稿は人が手動で行う（自動化しない）。

## 前提・方針

- 回答は**健全・無リンク・一発で笑える**もの（エロ/露骨/リンクはリプに混ぜない）。エロはプロフに来てから。
- **未成年連想ワード・実在個人の誹謗中傷・センシティブ時事ネタは禁止**（凍結/炎上リスクのみ）。
- 固定テンプレの使い回しはしない。**毎回その場でお題に合わせて新規に書く**。
- ノウハウは `ナレッジ/集客・導線/X運用ナレッジ_大喜利（坊主リプ集客）.md` に従う（実行前に必ず読む）。

## 入力

- ユーザーが渡す**お題（テキスト）**。例：「最悪な目覚まし時計とは？」「この写真に一言」。
- **写真お題**で画像が添付された場合は、画像を Read で見てから、その写真の“心の声 / 状況ナレーション”を作る。
- お題が渡されていない場合は、**お題を1つ聞いてから**作る（`AskUserQuestion` か短い質問）。

## 実行手順

1. **ノウハウを必ず読む**：`ナレッジ/集客・導線/X運用ナレッジ_大喜利（坊主リプ集客）.md`
   （10の型・伸びるリプの作法・安全ライン・1お題あたりの作り方）。
2. **お題を確認する**（写真お題なら画像も見る）。想定される**ベタな答えを1つ頭出し**して、それを避ける。
3. **複数の型で発想する**：ノウハウの「大喜利の型10」から**3〜4方向**で当てる
   （ズラし／誇張／あるある回収／例えツッコミ／擬人化／言葉遊び／裏切りオチ／具体の固有性／状況設定／共感×自虐）。
4. **強い順に3〜5本を仕上げる**。各案のルール：
   - **1行〜長くて2行**・口語・体言止め/言い切りでテンポよく。
   - **前提知識ゼロで笑える**（内輪・専門ネタは避ける）。絵文字0〜1個・ハッシュタグなし。
   - **健全・無リンク**。下品/露骨/未成年連想は禁止。
   - 各案に **使った型** と **ひとことの狙い**（なぜ刺さるか）を添える。
5. **イチオシを1本指名**する（いちばんリプが伸びそうなもの）と、その理由を一言。
6. （任意）保存を頼まれたら `fanza_auto/output/大喜利/<日付>.md` に、お題と回答案を追記する。

## 出力フォーマット（例）

```markdown
🪧 お題：最悪な目覚まし時計とは？

1. 鳴るたびに昨日の自分の発言を読み上げる
   └ 型：ズラし ／ 狙い：「最悪＝音量」のベタを外し、心理的ダメージで共感を取る

2. 二度寝した分だけ親に電話がかかる
   └ 型：誇張・極端化 ／ 狙い：実害のスケールアップで“それは無理”の笑い

3. 「あと5分」を言うたびにアラームが1時間ずつ早くなる
   └ 型：裏切り・落差 ／ 狙い：あるある（あと5分）から理不尽オチへ

⭐ イチオシ：1番。共感系は大喜利の場でいちばんいいねが付きやすく、プロフ流入を取りやすい。
```

## 完了後に必ず伝えること

- 出力は**候補**。リプは人が手動で、坊主などのお題に対して**1お題1〜2本**だけ選んで返す。
- **リプは健全・無リンク**を厳守。エロ/リンクの仕事は「プロフに来てから（bio・固定・メイン投稿）」。
- 新規期のリプ上限の目安は**〜20/日**。同じお題に連投しない・毎回ちがう文面・等間隔投稿はしない。
- 伸びた型は翌日少し寄せてよいが、**同じ回答の再利用はしない**。
