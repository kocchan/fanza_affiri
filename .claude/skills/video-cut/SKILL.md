# 動画カット（区間切り出し）

入力動画の「開始〜終了」区間を切り出して保存するスキル。
内部では `fanza_auto/cut_video.py`（ffmpeg ラッパー）を使う。

## 前提

- `ffmpeg` / `ffprobe` が入っていること（`which ffmpeg`）。
  無ければ `brew install ffmpeg` を案内・実行する。
- `fanza_auto/cut_video.py` が存在すること。

## 入力の確認

ユーザーから次を受け取る。曖昧なら聞き返す。

1. 入力動画のパス（例: `fanza_auto/output/<日付>/<名前>/sample.mp4`）
2. 開始時間
3. 終了時間
4. （任意）出力パス／ファイル名

時間は**秒数**でも **HH:MM:SS** でも指定できる。
- 秒数: `5` / `12.5`
- 時刻: `00:00:05` / `0:01:23.5`

## 実行手順

1. 入力動画が存在するか確認する（無ければパスを聞き直す）。

2. プロジェクトルートから次を実行する：

   ```bash
   python3 fanza_auto/scripts/cut_video.py "<入力動画>" <開始> <終了> ["<出力動画>"]
   ```

   例（5秒〜12秒）：
   ```bash
   python3 fanza_auto/scripts/cut_video.py "fanza_auto/output/2026-06-20/2026-06-20_朝_しずかちゃん/sample.mp4" 5 12
   ```

   例（時刻指定＋出力名）：
   ```bash
   python3 fanza_auto/scripts/cut_video.py "fanza_auto/.../sample.mp4" 00:00:05 00:00:18 cut.mp4
   ```

3. 出力先を省略すると、入力と同じフォルダに
   `<元の名前>_cut_<開始>-<終了>.mp4` が作られる。

4. 完了したら、出力パス・ファイルサイズ・実際の長さ（必要なら
   `ffprobe -v error -show_entries format=duration -of csv=p=0 <出力>` で確認）を
   ユーザーに報告する。

## 仕様メモ

- 音声は保持される（`-c:a aac`）。
- 再エンコード（libx264 / crf 20 / veryfast）するため、指定秒ピッタリで切れる。
  `-c copy` のキーフレームズレは起きない。
- `終了 <= 開始` の場合や入力が無い場合はエラーを出して止まる。
- 元動画は変更しない（常に別ファイルに出力）。

## よくある依頼の言い換え

- 「この動画の5秒から12秒だけ欲しい」→ 上記コマンド
- 「サンプルの美味しいとこだけ切って」→ 区間を聞いてから実行
- 「音声いらない／映像だけ」→ 必要なら `cut_video.py` の `-c:a` 部分を
  `-an` に変える運用も可（通常は音声込みで出す）
