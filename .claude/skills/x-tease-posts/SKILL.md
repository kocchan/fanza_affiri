# 文字だけの“引きのある”つぶやき生成（メイン育成）

アフィリエイト投稿（集客＋サブ引用）の**合間に挟む、文字だけの素のつぶやき**を毎日書き起こす。
画像もリンクも無いので**それ自体にリスクはほぼ無く**、メインアカウントの「人間味・エンゲージ・信用」を安全に積み上げるのが目的。

★方向性：上品にまとめすぎない。**Hあるある・夜の店あるある・「こういうことがあった」体験談**のように、
つい「わかる」「笑った」とリプしたくなる、共感と引きのあるものを狙う。

このスキルは**文章を作るだけ**。投稿は人が手動で、不定期に行う（自動化しない）。

## 前提・方針

- 固定テンプレの使い回しはしない。**毎回その場で新規に書く**（同一テキストの反復はシャドウバン要因）。
- 規制ライン・トーン・アーキタイプは `ナレッジ/コンテンツ/X運用ナレッジ_文字投稿（そそる）.md` に従う（実行前に必ず読む）。
- 入力データ・API は不要。

## 実行手順

1. **プレイブックと勝ち型を必ず読む**（2つとも）：
   - `ナレッジ/コンテンツ/X運用ナレッジ_文字投稿（そそる）.md` … トーン・8アーキタイプ・絶対NG（未成年連想/露骨描写/リンク）
   - `ナレッジ/コンテンツ/X運用ナレッジ_勝ち型（バズ型）.md` … 競合実測で裏取りした“伸びる型”。**記録/速報/ギャップ/断定/共感/問いかけ**を毎回最低数本は織り込む。

2. **被りを避ける**：`fanza_auto/output/つぶやき/履歴.md` があれば読む（無ければ新規）。
   直近の本文と**言い回し・構文・ネタが被らない**ようにする。

3. **今日のつぶやきを10〜12本書く**。ルール：
   - 8アーキタイプ（**Hあるある／夜の店あるある／体験談トーク**／性癖告白／問いかけ／時間帯あいさつ／自虐／におわせ）を**散らす**。同じ型の連発はしない。
     **前半の「あるある・体験談」系を主軸**にして、引きと共感を強める。
   - 必ず**引き（フック）**を1つ入れる：①具体的なワンシーンから始める ②「わかる人いる？/俺だけ？」で締める ③盛り上がり→現実の落差オチ ④「〜あるある」と言い切る。
   - 1本=1〜3行、口語、絵文字0〜2個、ハッシュタグは基本なし（たまに1個）。
   - **露骨・グラフィックな性描写はしない**（挿入の実況などNG。生々しさはシチュ・感情・落差で出す）。
   - **未成年を連想させる語は一切使わない**（学生/制服/JK/1●歳/ロリ等＝ゼロトレランス）。
   - **実在の店名・個人名・違法サービス誘導はしない**（夜の店ネタは一般論・自虐に留める）。
   - **URL・リンクは入れない**（無リンクが価値）。
   - 各本に「区分（アーキタイプ）」と「推奨時間帯（朝/昼/夜/深夜）」を添える。

4. **保存する**：`fanza_auto/output/つぶやき/<今日の日付>.md`（フォルダが無ければ作る）。
   フォーマット例：

   ```markdown
   # つぶやき案 2026-06-20

   > 文字だけ・無リンクの素のつぶやき。人が不定期に2〜4本だけ選んで、時間を散らしてメインから投稿する。
   > ⚠️ 未成年連想ワード・露骨描写・リンクは禁止。全部は使わない／同じ文の再投稿はしない。

   ## 1. ［共感・あるある／深夜］
   ```
   寝ようと思ってたのに、今日に限ってスイッチ入ってる。こういう日ほど明日が早い。
   ```

   ## 2. ［問いかけ・参加型／夜］
   ```
   夜にこっそり派？ 朝のほうが冴える派？ みんなどっち。
   ```
   …（10〜12本）
   ```

5. **履歴に追記する**：生成した本文を `fanza_auto/output/つぶやき/履歴.md` に日付つきで追記
   （次回以降の被り回避に使う）。

6. **報告**：生成した本数と、特に強い（エンゲージ取れそうな）2〜3本を抜粋してユーザーに見せる。

## 完了後に必ず伝えること

- 出力は**候補**。人の運用フェーズで以下を守る：
  - **全部は投稿しない**。その日の気分で2〜4本だけ選ぶ。
  - **不定期・バラバラの時間**に出す（機械的な等間隔はNG）。
  - **メインアカウントから**、画像・リンクを付けずに素の声として投稿する。
  - 反応が良かった型は翌日少し寄せてよいが、**同じ文の再投稿はしない**。
- このスキルは投稿しない。投稿は人が手動で行う。
