# DMM/FANZAアフィリエイト X運用支援

DMM/FANZAの動画をアフィリエイトで紹介するための、コンテンツ生成・運用ナレッジをまとめたプロジェクト。
**実際のX投稿は人が手動で行う**（このシステムは「投稿候補」を作るだけ）。

---

## 毎日の使い方

| やること | Claudeへの話しかけ方 |
|---|---|
| 今日の投稿素材を作る | 「今日の投稿を作って」 |
| 投稿ボードを開く | 「ボードを開いて」または `fanza_auto/works/board.html` をダブルクリック（コピー用途はこれで十分。切り抜き・アーカイブ・MissAV確認・URL取り込みは要サーバー） |
| MyFansのボードを開く | 「MyFansのボード開いて」（FANZAとは別の専用ボード。`myfans_auto/`） |
| 1作品を動画つきで開く | 「◯◯（cid）のボードを開いて」（再生・切り抜き・保存ができる） |
| 画像を選ぶ | 「画像を選びたい」（旧ボードを立ち上げる） |
| 文字投稿（つぶやき）を作る | 「今日のつぶやき作って」 |
| 大喜利の回答を考える | 「お題：◯◯ で大喜利考えて」 |
| 競合を分析する | スクショを `競合分析/` に入れて「競合分析して」 |
| 新アカウントを立ち上げる | 「新しいアカウントのロードマップ作って」 |

---

## フォルダ構成

| フォルダ | 役割 |
|---|---|
| `fanza_auto/works/` | **メインの作業場** — 作品ごとの投稿素材と投稿ボード（board.html） |
| `fanza_auto/output/` | つぶやきの保存先・使用済み作品の記録 |
| `fanza_auto/scripts/` | システムが自動で使うスクリプト（触らない） |
| `fanza_auto/_archive/` | 旧ボード一式（画像の切り抜き・A/B判定は今もここが担当） |
| `myfans_auto/works/` | MyFans専用の作業場と投稿ボード（FANZAとは別ボード。APIが無いためタイトル/説明文/サムネのみ自動取込、動画は拡張機能で保存→取り込みボタン） |
| `ナレッジ/` | X運用の知見（規制・安全 / コンテンツ / 集客・導線 / 立ち上げ） |
| `事業方針/` | 事業概要・運用方針（全体像を知りたいときに読む） |
| `競合分析/` | 競合バズ投稿のスクショ＋分析レポート |
| `ロードマップ/` | アカウント立ち上げロードマップHTML（コンセプト別） |

## 規制の大原則（必読）

- 未成年に見える素材・制服強調・JK等のワードは**絶対に使わない**（発覚したら即凍結）
- メインアカウントにアフィリエイトリンクを直接貼らない（サブ→メイン引用が導線）
- 詳細は `ナレッジ/規制・安全/` を参照
