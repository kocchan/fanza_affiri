# X運用ナレッジ：文字だけの“引きのある”つぶやき（メイン育成）

最終更新：2026-06-20

メインアカウントを「人間味のある強いアカウント」に育てるための、**文字だけ（画像・リンクなし）の引きのあるつぶやき**の型をまとめる。
アフィリエイト投稿（メイン集客＋サブ引用）だけだと、アカウントが「宣伝botっぽく」見えてリーチも信用も伸びない。
その合間に、**思わず反応したくなる・共感する・笑えるつぶやき**を不定期に挟むことで、人間らしさとエンゲージを積み上げる。

> ★方向性（重要）：上品にまとめすぎない。**「Hあるある」「夜の店あるある」「こういうことがあった体験談」**のような、
> ちょっと生々しくて共感できて、つい「わかる」「笑った」とリプしたくなるものを狙う。

---

## 目的（なぜ文字投稿が必要か）

| 狙い | 中身 |
|---|---|
| **① 人間味** | 宣伝以外のつぶやきがあると「中の人がいる」感が出る → 凍結審査でもbot判定されにくい |
| **② エンゲージ** | 共感・あるある・問いかけはリプ/いいね/RTが付きやすい → アルゴリズム評価が上がりメイン全体のリーチが伸びる |
| **③ 信用の貯金** | 無リンク・無画像の素の投稿はリスクゼロ。これを日常的に出してアカウントの健全度を底上げ |
| **④ 集客投稿の下地** | 「この人の下ネタ面白い／わかる」と思わせておくと、後の集客投稿（画像/動画つき）の刺さりが上がる |

---

## 引き（フック）の作り方 ★ここが一番大事

ただのエロいつぶやきは流される。**「反応したくなる仕掛け」**を必ず1つ入れる。

1. **具体的なワンシーンから始める**（抽象的な感想で始めない）。
   - ✗「夜って欲が出るよね」 → ◯「いい雰囲気の真っ最中にスマホの通知音鳴って、二人で真顔になった」
2. **共感を取りにいく締め**：「わかる人いる？」「俺だけ？」「これ共感してくれ」でリプを誘う。
3. **落差・オチ**：盛り上がり→現実、で笑わせる（賢者タイム・ゴム探し・通知音 等）。
4. **あるある化**：「〜あるある」「〜な瞬間、絶対ある」と言い切ると保存・引用されやすい。
5. **言い切る／本音**：かっこつけず、しくじりや本音を出すほど刺さる。

## 投稿アーキタイプ（引き出し）

毎回これらを**散らして**使う（同じ型ばかり連投しない）。前半4つ（あるある／体験談系）を主軸に。

1. **Hあるある** … 性のリアルな共感。賢者タイム・声我慢・終わった後の塩対応・段取りの萎え。
   - 例:「終わった瞬間に世界一冷静になるの、あれ毎回謎。さっきまでの俺どこ行った。」
   - 例:「いい雰囲気なのにゴム探して部屋ひっくり返す時間、毎回いちばん萎えるしいちばん笑う。」
2. **夜の店あるある** … キャバ/ガールズバー/出会い系などの“あるある・失敗談”。固有店名・個人名は出さない。
   - 例:「源氏名ど忘れして『えっと…君！』で乗り切ろうとした夜、一生忘れない。」
   - 例:「緊張しすぎて会話が面接みたいになるの、夜の店あるあるじゃない？俺だけ？」
3. **体験談トーク（〜なことがあった）** … 「この前さ…」で始まる短いエピソード。オチ・共感・笑い。
   - 例:「この前、ムード最高潮で『あ、明日燃えるゴミだ』って急に思い出して集中力ゼロになった。」
4. **性癖・こだわり告白** … ギャップ・シチュ萌え。「わかる人だけ」でリプ誘発。
   - 例:「派手な子より“普段おとなしいのに脱ぐと…”のギャップに一生勝てない。わかる人いる？」
5. **問いかけ・二択（参加型）** … リプ/引用を稼ぐ。
   - 例:「夜にこっそり派？ 朝のほうが冴える派？ みんなどっち。」
6. **時間帯あいさつ＋ひとネタ** … おはよう/おやすみに軽い下ネタ。人間味＋投稿の規則性をぼかす。
   - 例:「おやすみ。…と言いつつ、たぶんあと1本だけ見る。みんなもほどほどに夜更かしして。」
7. **自虐・しくじり・散財** … 笑える方向のエロ。
   - 例:「気づいたら今月も“沼”に課金してた。後悔はしてない、反省もしてない。」
8. **におわせレビュー／好奇心フック** … 作品名もリンクも出さず雰囲気だけ。後の集客への布石。
   - 例:「さっき見たやつ、内容より先に表情に記憶持っていかれた。最終的に“顔”なんだよな。」

## ⚠️ 絶対NG（文字でも一発アウト・凍結級）

X のゼロトレランスはテキストにも適用される。以下は**言葉だけでも絶対に書かない**。

- **未成年を連想させる一切の語**：`学生` `JK/J系` `制服` `女子高生` `1●歳` `ロリ` `幼い` 等。
  実演者が成人でも、Xは「未成年に見える/連想させる」だけでゼロトレランス（永久凍結＋通報）。
- **露骨・グラフィックな性行為の実況描写**（挿入の具体描写など＝テキストAdult/Explicit相当）。
  あくまで**あるある・におわせ・笑い・共感**に留める（生々しさはシチュや感情で出す）。
- **非同意・近親・獣姦などの違反テーマ**、盗撮・無断などの匂わせ。
- **実在の店舗名・個人名／違法サービス（売春・援交等）への誘導**。「夜の店」ネタは一般論・自虐に留める。
- **リンク・URL**（このスキルの投稿は無リンク。URLは入れない）。

## 投稿の運用（人が手動でやること）

- **不定期に散らす**。1日2〜4本を、朝・昼・深夜などバラバラの時間に。機械的な等間隔はNG。
- **生成した全部は使わない**。その日の気分で数本だけ選ぶ。
- **メインアカウントから**素の声として投稿する（サブではない）。
- 画像・リンクは付けない（付けた瞬間、性質が「集客/宣伝」に変わる）。
- 伸びた型（どのあるあるが刺さったか）は覚えておき、翌日以降に少し寄せる（丸パクリ再投稿はしない）。
