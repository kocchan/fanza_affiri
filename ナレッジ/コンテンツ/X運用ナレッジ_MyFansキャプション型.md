# X運用ナレッジ — MyFansキャプション型

MyFansダッシュボード（`myfans_auto/`）のメイン投稿文を作る `myfans_auto/scripts/caption.py` が
参照する「型」の元ネタ。FANZA側の `X運用ナレッジ_勝ち型（バズ型）.md` に相当するMyFans版。

## 考え方

MyFansには評価・ジャンル・発売日のような構造化データが無く、あるのは投稿ページの自由記述の
本文（description）だけ。なので「型」は評価の高さではなく、**単語（場所・人物属性・イベント・
ゲーム等）を短い反応文のテンプレートに当てはめる**方式で作る。

- 本文からの自動抽出（`extract_situation`）は**場所・シチュエーション**の単語だけを許可リストで拾う
  （エアビー・お泊まり・ホテル・合コン・デート等）。行為語・ハッシュタグ・未成年連想語は
  本文にあっても絶対に拾わない。
- ボードの「🔄 AI再生成」ボタンは、人が自由に入力した単語を使う（許可リストを介さない＝
  人が安全な単語を選ぶ前提）。**人物属性やイベント名などプレースホルダー1つで成立する単語なら
  何でも使える**（下の実例参照）。

## 実例（ユーザー提示・2026-07-23）

以下はユーザーが「こういう文章を参考にしてほしい」と提示した実例。`TEMPLATES` の型はこれらの
構造を一般化したもの。

```
インドア美少女との密会がこれ
カーストトップの女子に目をつけられるとこうなる
同窓会で久々に再会したらこうなるｗｗ
まだW杯気分の女さんｗｗｗｗ
野球拳で負けちゃう〇学校教師さんｗｗｗ
```

**共通する型の要素**：
- 短い1〜2文で完結（CTA行・ハッシュタグは付けない）
- 語尾に「www」「ｗｗ」で軽さ・自虐/あるある感を出す（無くてもよい＝断定調で終わるパターンもある）
- 単語（場所／人物属性／イベント／ゲーム）を1つ主役に置き、「〜がこれ」「〜こうなる」
  「〜さんｗｗｗ」で落とす
- 人物属性（インドア美少女・カーストトップの女子・〇学校教師）は**職業・キャラ設定であり、
  年齢を主張しない**（学校教師＝成人の職業。生徒/学生側の属性ではない点に注意——未成年連想語
  （女子高生・JK等）は`post_text.py`のSAFE_GENRES同様ここでも使わない）

```
{kw}なう→気づいたらこうなってましたwww
男女で{kw}したらそりゃこうなるwww
{kw}の誘惑に勝てる人いる？www
{kw}に誘われたら断れない自信あるwww
{kw}との密会がこれ
{kw}に目をつけられるとこうなる
{kw}で久々に再会したらこうなるｗｗ
まだ{kw}気分の女さんｗｗｗｗ
{kw}で負けちゃう子ｗｗｗ
```

新しいパターンを思いついたら、まずここに実例として追記してから `caption.py` の
`TEMPLATES`/`SAFE_SITUATIONS` に反映する（このファイルが元ネタ・スクリプトが実装、の関係を保つ）。
