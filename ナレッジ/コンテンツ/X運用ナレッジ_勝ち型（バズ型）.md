# X運用ナレッジ：勝ち型（バズる投稿の型）

最終更新：2026-06-21
出典：競合「SNSでは言えない話」実測分析 → [競合分析_SNSでは言えない話.md](../../競合分析/競合分析_SNSでは言えない話.md)

> ★このファイルは **`x-tease-posts` と `fanza-content` の文面を作るときに必ず参照する**こと。
> 競合の実数字（11万/5.8万インプ）で「効いた型／外した型」が裏取りされた“勝ちパターン”。

---

## 結論：伸びる＝フック、伸びない＝ただのエロ

実測でハッキリ出た差：
- **伸びた**：記録・ランキングの社会的証明＋速報＋複数枚 → **11万・5.8万インプ**
- **外した**：フックの無いグラビア/水着だけ → **836・286インプ**

→ **「何を見せるか」より「どう一言で引くか」で桁が変わる。** 毎投稿に必ず下の型を1つ入れる。

---

## 勝ち型 6パターン（優先度順）

1. **記録・ランキングの社会的証明（最強）**
   - 「日間・週間・月間 連続1位」「年間No.1」「○○でも出来なかった偉業」
   - 数字・順位で“みんなが選んでる安心”と興味を同時に作る。
2. **速報体**：頭に「【速報】」。今すぐ感・ニュース性。
3. **ギャップ提示（2枚並べ）**：1枚目で油断（きちんとした雰囲気）→2枚目で落とす。
4. **断定・煽り**：「今後10年は出てこない」「○○すぎる」。言い切ると引用される。
5. **共感・自虐**：「語彙力なくなった」「腱鞘炎なったわ」。人間味でリプを誘う。
6. **問いかけ（参加型）**：「どっち派？」「わかる人いる？」。返信＝最重要エンゲージ。
7. **ランキング＋伏せ＋スレ連結（新・最強級）**：「今年バズった〇〇ランキング」で 3位→2位→**「1位はさすがに…」で止める**。
   - 答え（1位）・動画・リンクは**本文に書かず、リプ（スレ）に置く**。好奇心ギャップでブックマーク／リプ／滞在が同時に伸びる。
   - 実測：競合「業界人M」が本投稿だけで **1.1M（110万）インプ**（既存の11万/5.8万をさらに1桁更新）。出典 → [競合分析_業界人M.md](../../競合分析/競合分析_業界人M.md)
   - メイン無リンク原則と完全両立（本文で外部誘導しない＝初速が死なない）。「1位だれ？」を当てさせて**自動で参加型**に転化する。
   - **うちの実装**：**2位/3位＝有名女優（固定リスト `fanza_auto/famous_actresses.json` ＋顔写真）／1位＝本命の素人を“伏せて”動画＋リンクで回収**。
     「有名どころを抑えて1位はまさかの素人」という**意外性**でクリックを取り、リンクは素人作品（正規FANZA）へ。
   - ⚠️ **「個人的ランキング（主観）」として出す**（断定の事実主張・名誉毀損にしない）。未成年連想・制服強調は厳禁、センシティブON。

### 形式の鉄則
- **1行キャプション＋複数枚**（4分割 or 2枚ギャップ）が基本。
- 絵文字は少量（💦など0〜2個）、`www` `〜` のゆるい口語。
- **表示名自体にフック**を仕込む（競合「SNSでは言えない話」＝禁断。うちは「夜の当たり研究所」＝目利き）。
- **メインは無リンク**。リンクはサブ→引用で（[導線設計](../集客・導線/X運用ナレッジ_導線設計（競合分析）.md)）。
- 投稿後30〜60分はリプ対応できる時間に出す（[インプ向上](../集客・導線/X運用ナレッジ_インプ向上.md)）。

---

## ⚠️ 盗むのは「型」だけ。題材の危険は盗まない

競合は **制服・学生強調／「脱法ロリ」等の未成年連想ワード／MISSAV(海賊版)参照** で稼ごうとしているが——
- これは **Xゼロトレランス（永久凍結＋通報）**。しかも該当投稿は **286インプと伸びてすらいない**。
- うちは **記録/速報/ギャップ/断定の“フック構造”だけ**を移植し、題材は **明らかに成人・大人っぽい“当たり作品”** に限定。
- 未成年連想ワード厳禁・露骨フレーム不使用・正規FANZAのみ（[アダルト規制](../規制・安全/X運用ナレッジ_アダルト規制.md)）。

## そのまま使える文例（規制セーフ・無リンク）

### 集客投稿向け（fanza-content / 画像・動画つき）
- 〔記録〕`日間・週間・月間ぜんぶ上位。“当たり”の中の当たりだわ。`
- 〔記録〕`ランキング上位なの、見て一発で納得した。数字は嘘つかない。`
- 〔速報〕`【速報】これ今年のトップ争い、完全に入ってきた。`
- 〔断定〕`今年これ超えるの、当分出てこない気がする。`
- 〔断定〕`これ知らずにスルーは、普通に損してる。`
- 〔ギャップ〕`この“きちんとした雰囲気”で…ってのが一番ずるい。`（2枚並べ）

#### ★ランキング＋伏せ＋スレ連結（メイン無リンク／実名は使わない）
- メイン：`今週“当たり”だった作品ランキング。3位…◯◯系、2位…◯◯系。1位はさすがに言葉にできなかった。`
- メイン：`タイプ別「沼る」ランキング。3位 清楚系 / 2位 ギャップ系 / 1位は…続きは下に置いとく。`
- 1つ目リプ（続き・動画）：`1位、これです↓（動画）` / `続き↓ 見たら理由わかると思う。`
- 2つ目リプ（サブ→リンク）：`気になった人用に置いとく↓【サブの引用 or リンク】`

### つぶやき向け（x-tease-posts / 文字だけ）
- 〔共感〕`良すぎて語彙力なくなった。誰か語彙返して。`
- 〔記録ネタ〕`ランキング上位、伊達じゃないんだよな。並んでるやつ全部わけが違う。`
- 〔問いかけ〕`記録ずくめの話題作と、隠れた良作。最初に見るならどっち派？`
- 〔速報風〕`【自分内速報】今年の上半期ベスト、更新されたかもしれない。`

## ★伸ばしの3点セット（2026-07-19 追記・実装反映済み）

実運用フィードバックで効くと分かった3点。fanza-content の投稿文（`post_text.py` / `templates.py`）に反映済み。

### ① 伏せ字＋“驚き実況”で刺す（短く・規制を避ける）
Xの検索/規制に引っかかりにくくしつつ連想させる。**露骨に書かず伏せ字**＋「こんなに可愛い子が◯◯」の
**驚き・感情の実況トーン**が効く（2026-07-19 ユーザー方針）。
- 伏せ字：`A▽`（AV）／`エ□`（エロ）／`セッ久`。
- 文例：
  - `こんなに可愛い子がA▽デビューしてるの、令和のバグすぎる。`
  - `こんな子のセッ久が見れる時代、まじで感謝しかない。`
  - `この顔でセッ久してくれるの、控えめに言って事件。`
  - `清楚なのにエ□い、この温度差。`
- ⚠️ A▽（＝AV）は**作品そのもの**。「A▽デビュー」「A▽側に来た」等の事実で使う。
  「A▽超え」等の比較はNG（この作品自体がAVなので意味が通らない）。
- ⚠️ 伏せ字でも**未成年連想ワードは入れない**（学生/JK/ロリ等）。「可愛い子」は**成人前提**でのみ。
  行為の露骨描写はしない（Sensitive止まりの“におわせ”）。
- プールは `templates.py` の `MASK_HOOKS`。足すときは必ずここ。

### ② “似てる/系”比較で刺す（★実際のAV女優名を使う）
「○○に似てる」「○○系」は刺さる。**必ず実際のAV女優の名前で出す**（“っぽい/系のタイプ表現”は
ぼやけて弱いので使わない）。名前の出どころで安全性が真逆になる点に注意。
- ✅ **AV女優（同業）の実名**＝これを使う。`fanza_auto/famous_actresses.json` に列挙。“話題の”女優を
  足すほど刺さる＆重複しない。低リスク・業界の定番。
  - 文例：`{name}に似てるって絶対言われる。` / `顔、{name}系。` / `雰囲気が完全に{name}。`
- 🚫 **一般芸能人・アイドルの実名＝使わない（ゼロトレランス）**。肖像権・パブリシティ権・名誉毀損・
  X規約違反＝**凍結＋法的クレーム**。CLAUDE.md「実在個人の中傷」に正面から反する。
- タイプ表現（`COMPARE_TYPE`）は**女優名リストが空のときのフォールバックだけ**。通常は実名を出す。
- ※「当たり」という言い回しは使わない（ユーザー方針 2026-07-19）。

### ③ 短くパッと分かる（2〜3行）
だらだら書かない。**伏せ字フック → 刺さる一言（比較 or レビュー数字）→ 短いCTA** の3行が基本形。
- 実装：メイン本文は約30〜50字。ハッシュタグは付けない（長くなる／機械的に見える）。
- 例）
  ```
  顔よし雰囲気よし、A▽級。
  三上悠亜に似てるって絶対言われる。

  気になる人は引用元へ👇
  ```
