## ★短い一文リアクション調

「短い一文の方がウケがいい」
`templates.py` の `REACTION_HOOKS` に採用し、`post_text.py` の `build_main_text()` を
**フック1文＋CTAだけ**（従来の「フック＋具体1行＋CTA」から具体行を削除）に短縮した。

- 使える例：`これがちょうど良すぎて毎日お世話になってる` / `発射させた人が勝ちらしいwww` /
  `え、めっちゃかわいくない？` / `清楚系美女のギャップが凄まじすぎるwww`
- 作品データ（レビュー文章・説明文）を絡めた動的な一文も生成する（`post_text.reaction_line()`）：


“似てる/系”比較（AV女優名を使う型）はこの変更で廃止した（`compare_line`/`COMPARE_NAME`/
`COMPARE_TYPE` を削除）。短い一文優先の方針と競合するため。復活させたい場合は
`post_text.py`/`templates.py` の git 履歴を参照。
