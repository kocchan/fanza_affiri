# X運用ナレッジ：アルゴリズム解析（2026年版）

最終更新：2026-06-21
出典：2026年版 Xアルゴリズム解析レポート（外部）を、うちのアダルト運用向けに抽出・翻訳。
位置づけ：[X運用ナレッジ_インプ向上.md](./X運用ナレッジ_インプ向上.md) の“数値の裏取り”版。判断に迷ったらこの数理に従う。

> 前提：Xは「フォロー（誰と繋がるか）」より「おすすめ For You（何が面白いか）」主導＝**フォロワー0でもコンテンツの質と初速で数千〜数万インプは出せる**。

---

## 1. エンゲージのスコア重み（★これが全て）

「いいね」を1倍とした相対価値。**会話と滞在時間が桁違いに強い**。

| アクション | 倍率/点数 | 解釈 |
|---|---|---|
| **自分が返信（リプ往復）** | **+75点 / 150倍** | 最強。投稿後に自分で返信＝会話発生シグナル |
| 他者からのリプ | +13.5〜27点 / 27倍 | 能動的に文字入力させた価値 |
| プロフクリック＋反応 | +12点 / 24倍 | 発信者本人への興味（インタレストグラフ） |
| 詳細クリック（タップ/拡大/さらに表示） | +11点 | もっと読もうとした意欲 |
| **2分以上の滞在（dwell）** | **+10点 / 20倍** | アテンション経済の最重要指標 |
| ブックマーク | +1.5点 / 3倍 | 保存＝長期資産コンテンツ |
| リポスト | +1.0点 / 2倍 | 拡散 |
| いいね | +0.5点 / 1倍 | 最弱。単体では伸びない |

→ **結論：質の高い1投稿 ≫ 凡庸な10投稿。** 「いいね数」より「リプ往復・滞在時間」を取りにいく設計にする。

## 2. ネガティブは桁違いに重い（事故ると即死）

| アクション | 倍率/点数 | 取り返すのに必要な“いいね” |
|---|---|---|
| 通報 | -369〜-738点 / -100倍 | **約738件** |
| ブロック | -148点 / -50倍 | 148件 |
| ミュート | -74〜-89点 / -30倍 | 約89〜148件 |
| 「興味なし」 | -59点 / -20倍 | 59件 |

→ **通報1回＝いいね700超を吹き飛ばす。** 炎上・過度な煽り・スパム的タグ乱用は長期的にアカウントを破壊。
うちのアダルトは特に**未成年連想・露骨表現での通報が致命傷**（[アダルト規制](../規制・安全/X運用ナレッジ_アダルト規制.md)）。不快を与えない運用が絶対条件。

---

## 3. バイラル・ウィンドウ：勝負は投稿後2時間（特に最初の15分）

1. **0〜15分**：フォロワーの10〜20%に試験配信（想定100〜5,000imp）。
2. **15〜60分**：最初の15分の**エンゲージ率が3%超**なら50〜80%へ拡大（5〜10倍）。**1%未満なら配信停止**。
3. **1〜2時間以降**：高評価維持で**For You（フォロー外）に放流→100〜1,000倍**に爆発。

### 初速ハック（最重要・うちでも即効）
- 投稿に**オープンな問い／アンケート**を仕込み、リプを誘発。
- **投稿後1〜2時間は張り付いて、来たリプに即返信**（リプ往復＝150倍を量産）。「ありがとう」一言でも加点。
- 初期はリプが付きにくい → **同ジャンルの他者へ自分から気の利いたリプ／引用**で巻き込む。
- → うちの `x-tease-posts`「あるある＋問いかけ」はこの初速設計そのもの。

## 4. 投稿のゴールデンタイム
- 7〜9時（通勤）／12〜13時（昼）／18〜19時（帰宅中）／**20〜22時（最強・帰宅後）**。
- ターゲットが起きてない時間に投げるのは初速の自殺。アダルトは深夜帯も相性良。

---

## 5. 滞在時間を伸ばすコンテンツ設計
- **画像/動画で約2倍ブースト**。中でも**ネイティブ動画が最優遇**（15〜60秒、最初の3秒で掴む、無音でも分かるテロップ）。→ `fanza-content` の sample.mp4 を優先する根拠。
- **画像は2〜4枚**で構成（スワイプ＝詳細クリック+11／滞在+10を稼ぐ）。→ 競合の「4分割画像」が伸びてた裏付け（[勝ち型](../コンテンツ/X運用ナレッジ_勝ち型（バズ型）.md)）。
- **冒頭1〜2行に強烈フック**→「さらに表示」をタップさせる＝詳細クリック加点。スレッド形式で読ませると滞在↑。
- **ブックマーク誘導**（「保存推奨」）でブックマーク率を上げる（いいねの3倍）。

## 6. 新規アカウントの初期防衛（最初の7〜10日）

ボット判定（初期シャドウバン）を避けるための行動上限。**うちの新規メインはまずこれを死守**。

| 行動 | 1日の上限/注意 |
|---|---|
| 投稿 | **3〜5件**まで。短時間の連投はNG（Author Diversity Scorerで自己スパム減点） |
| リプ | 10〜20件程度。数分ごとの連打NG |
| いいね | 50〜80件まで。連打NG |
| フォロー | 数十分で数十件は即サーチバン級。少しずつ |
| 外部リンク | **本投稿に貼らない**。リンクは自己リプ/サブ→引用に隔離（[導線設計](./X運用ナレッジ_導線設計（競合分析）.md)） |

### シャドウバンの判定サインと回復
- サイン：①ログアウト/別垢で自分の投稿が検索に出ない ②リプが「その他の返信」に隠れる ③インプが常に1桁〜十数で張り付く。
- 回復：**焦ってプロフ/ID変更・フォロー解除を繰り返さない**。**24〜48時間（重度は3日〜1週間）完全放置**でスパム疑い自然解除を待つ。興味関心チェックを外すと早まる例も。

---

## 7. アカウント設計（X-SEO・ジャンル統一）
- **発信軸を1ジャンルに固定**（アルゴが「何者か」を学習＝関心度評価UP）。うちは「**成人・当たり作品の目利き＋夜のあるある**」に統一。
- **表示名/プロフィールに検索キーワード**を自然に入れる（うちは「夜の当たり研究所＝目利き」が該当）。
- ハッシュタグは**1〜2個の少数精鋭**。無関係トレンドタグ乱用＝キーワードスタッフィング扱いで減点。
- 投稿比率の目安：**有益7：共感3**。うちなら「有益＝当たり紹介/目利き」「共感＝Hあるある・夜の店あるある」。

## 8. X Premium（課金ブースト）— うちは“慎重に”
- 効果：投稿がフォロー外に**約2倍**・フォロワーに**約4倍**おすすめ表示。リプも上位表示。長文/太字も可。
- レポートは「必須インフラ」と推奨。**ただしうちはアダルト×新規**なので：
  - まず**初期7〜10日のトラスト構築を優先**（[インプ向上](./X運用ナレッジ_インプ向上.md)の方針）。
  - アダルト運用での課金は**アカBANリスク時の損失も上がる**ため、**健全に育って手応えが出てから**検討。
  - → 「強力な加速材だが、最初の保険ではない」という位置づけ。

---

## まとめ：うちの運用への落とし込み（優先順）
1. 初期7〜10日は**行動量を絞り**外部リンクを本投稿に貼らない（防衛）。
2. **ゴールデンタイム（特に20〜22時/深夜）**に投稿。
3. **問いかけ＋投稿後2時間の即リプ**で「リプ往復150倍」を量産（初速突破）。
4. **動画優先・画像2〜4枚・冒頭フック**で滞在時間を稼ぐ。
5. **通報を絶対に食らわない**（未成年連想/露骨/煽りすぎを排除）。
6. 余力が出たら Premium / 過去バズのリライト再投稿で底上げ。
