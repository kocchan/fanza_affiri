# X運用ナレッジ：導線設計（メイン×サブの引用RT動線）

最終更新：2026-06-20

競合の実運用から学んだ、メイン/サブの最適な導線設計のナレッジ。

---

## 観察した競合の構造

- **メイン「ゆん」(@mourinho47)**：「ガチ逸材すぎてバズってるやつ」＋画像/動画 → **6.3万表示**の集客投稿。**本体にリンクは無い**
- **サブ「人生」(@domarkou)**：「🔥ガチで神作🔥 続きは👇👇👇」＝**アフィリエイトリンクを持つ投稿**
- **メインが、そのサブの投稿を「引用」してスレッドにぶら下げている**

```
メイン本投稿（集客・無リンク・高リーチ）
   └─ メインがサブ投稿を【引用RT】でスレッドに追加
         └─ 引用先のサブ投稿に「続きは👇」＋アフィリエイトリンク
               └─ ユーザーがタップ → サブ投稿 → リンク → 購入
```

---

## なぜ「サブを引用RT」する形なのか

| 理由 | 中身 |
|---|---|
| **① メイン本体にリンクを残さない** | リンクは reach を抑制＋信用を削る。メイン投稿は無リンクで純粋に伸ばし、リンクはサブ側に隔離する |
| **② メインの集客力をサブに"輸血"する** | メインを見た数万人に引用カードが目に入る → タップでサブ投稿 → リンクへ誘導。**メインのリーチでサブのリンクを露出させる**のが核心 |
| **③ サブが飛んでもメインは生き残る** | リンクを持つ＝凍結リスクが高いのはサブ。捨て駒をサブにし、資産（メイン）を守る |
| **④ 自己宣伝に見えない** | 自分でリンクを貼らず「他人の投稿を引用・紹介」する形 → アルゴリズム的に自己宣伝スパムに見えにくい |
| **⑤ 社会的証明** | 「バズってるやつ」「ガチで神作」＝既に話題、という体裁。引用すること自体が"盛り上がってる感"を演出する |

## 重要な学び

### 「メインがサブを引用RT」＞「サブがメインにリプライ」
引用RTのほうが動線が強い：
- スレッド内で**独立した目立つカード**として表示される
- タップ動線が明確（引用カード → サブ投稿 → リンク）
- メインの投稿文を汚さない

### リンク資産は使い回す
- サブの元投稿は **6/6**、メインの引用は **約21時間前**。
- **古いサブ投稿を新しく引用し直して再露出**させている。
- → 1本のリンク投稿を作れば、メイン側で何度も引用して使い回せる。

### フックの型
- 「ガチ逸材すぎて**バズってるやつ**」＝ 既にバズっている体（社会的証明）で興味を引く
- サブ側は「🔥ガチで神作🔥 **続きは👇👇👇**」＝ 続きへの誘導を明示

## 自分の運用への落とし込み

投稿の順番を以下の3ステップにする：

1. **サブ**：アフィリンク付きの「続きは👇」投稿を先に作っておく（使い回し可能）
2. **メイン**：集客用の本投稿（無リンク・画像/動画・バズ風フック）を出す
3. **メイン**：1のサブ投稿を**引用RT**でスレッドにぶら下げる

→ メインの信用とリーチを守りつつ、リンク露出を最大化できる。
