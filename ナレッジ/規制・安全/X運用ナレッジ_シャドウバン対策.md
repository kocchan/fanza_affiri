アカウント# X運用ナレッジ：シャドウバンの条件（レイヤー別）

最終更新：2026-06-20

X 運用で回避すべきシャドウバンのリスク要因を、レイヤーごとに整理する。

---

## インフラレイヤー
- IP アドレスがブラックリストに登録されていないか
- 同一 IP からの複数アカウント瞬時切り替え

## アカウントレイヤー
- **アカウントエイジング**：作成直後（1〜2 週間）のアカウントが URL を貼るのは NG
- メールアドレス認証 / 電話番号認証は行う

## アクションレイヤー
- 機械的なタイミングでの投稿
- URL の比率の多さ
- 急激なアクティビティ

## コンテンツレイヤー
- 全く同じテキスト / 画像 / URL の使い回し
- センシティブ設定忘れ。そもそも肌の露出の多い投稿は控える
- 「スパム」「ブロック」「ミュート」を他アカウントから食らう
